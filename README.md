# Transport Groups Pilot App

A small Vite + React prototype for the first Transport Groups pilot.

## Run locally

```bash
npm install
npm run dev
```

## Deploy on Vercel

1. Create a GitHub repository and upload these files.
2. In Vercel, choose **Add New Project**.
3. Import the GitHub repository.
4. Accept the default Vite settings.
5. Click **Deploy**.

## What this version includes

- Owner onboarding
- Car and NFC tag setup
- Tap in / tap out page
- Demo trip state
- Simple event recording in local state
